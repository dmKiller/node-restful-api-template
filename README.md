#说明

* 下载后安装包 #npm install
* node ./bin/www
* -----------------或者-----------------
* 安装supervisor CLI： npm install -g supervisor
* 安装完后本地启动热更新CLI： supervisor bin/www

* 访问127.0.0.1:3000
